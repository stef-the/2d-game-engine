# Controls

You can set custom controls in `/src/lib/data.js` inside the controls variable.

```js
export const controls = {}
```

The default controls are as follows:
| **Purpose** | **Keybind** |
| Move Forwards | `w` |
| Move Backwards | `s` |
| Move Left | `a` |
| Move Right | `d` |
| Open/Close Debug Menu | `m` |
| Toggle Freeze State | `f` |
