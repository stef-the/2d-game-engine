# Controls

You can set custom controls in `/src/lib/data.js` inside the controls variable (example below).

```js
export const controls = { moveUp: "w" }
```

The default controls are as follows:

| **Action** | **Key** |
|-----|-----|
| Move Forwards | `w` |
| Move Backwards | `s` |
| Move Left | `a` |
| Move Right | `d` |
| Open/Close Debug Menu | `m` |
| Toggle Freeze State | `f` |
